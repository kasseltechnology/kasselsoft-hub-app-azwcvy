
# KasselHub - KasselSoft Mobile App

A professional, bilingual (Arabic/English) mobile application for KasselSoft - Information Technology & AI Development Company.

## Features

- **Bilingual Support**: Full Arabic (RTL) and English (LTR) support with automatic language detection
- **5 Main Sections**:
  - Home: Company overview, featured services, and Mr. Kassel AI trainer
  - Services: Comprehensive IT & AI solutions
  - Projects: Case studies and success stories
  - Blogs: Latest insights and updates
  - Contact: Jordan and UAE office information with direct call/WhatsApp integration

- **Professional Design**: Blue-to-purple gradient theme inspired by the Kassel logo
- **Dark Mode**: Full support for light and dark themes
- **Native Performance**: Built with React Native and Expo 54

## Tech Stack

- React Native 0.81.4
- Expo 54
- TypeScript
- expo-localization for i18n
- expo-linear-gradient for brand gradients
- React Navigation for routing

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Run on iOS:
   ```bash
   npm run ios
   ```

4. Run on Android:
   ```bash
   npm run android
   ```

## Language Switching

The app automatically detects the device language. Users can manually switch between Arabic and English using the language toggle button in the header.

**Note**: When switching between RTL (Arabic) and LTR (English), the app will reload to apply the text direction changes.
